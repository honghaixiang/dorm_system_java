# 🎓 学生宿舍管理系统

> 一个功能完整、界面华丽的Web端宿舍管理系统

## 🚀 快速启动

```bash
# Linux/Mac
./start.sh

# Windows  
start.bat
```

访问: http://localhost:8080/login.html

## 🔑 默认账号

**管理员:** admin / admin123

## ✨ 核心功能

### 学生端
- 注册登录
- 宿舍信息查询
- 报修申请
- 费用缴纳
- 活动报名
- 公告查看

### 管理员端
- 宿舍管理
- 学生分配
- 维修工单
- 费用管理
- 活动管理
- 公告发布
- 数据统计

## 💻 技术栈

- 后端: Java + Spring Boot + MyBatis
- 前端: HTML5 + CSS3 + JavaScript
- 数据库: H2
- 特色: 响应式设计 + 现代化UI
